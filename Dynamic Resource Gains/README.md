## ✨ Features

- ✅ **Dynamic scaling** for Science, Funds, and Reputation based on player progress
- ✅ **Toolbar button** available only in Space Center (Career Mode only)
- ✅ **In-game Manual** with detailed explanations and examples
- ✅ **Current Status Display** showing real-time penalties
- ✅ **Apply and Save workflow** - test without saving!
- ✅ **Fully customizable** - change any parameter
- ✅ **Reset to Default** button for quick reset

---

## 🎯 How It Works (30 seconds version)

**Dynamic Science:** As you unlock more tech tiers and gain more lifetime science, you get **less science** from missions (to balance the game).

**Dynamic Funds:** As you earn more total funds, you get **fewer funds** from contracts (to balance economy).

**Dynamic Reputation:** As you earn more total reputation, you get **less reputation** from missions (to balance progression).

You control how aggressive these penalties are!

## 🚀 Quick Start

### Install

```text
KSP\GameData\DynamicResourceGains\Plugins\Dynamic Resource Gains.dll
```

### Use

1. Start KSP in **Career Mode**
2. Go to **Space Center**
3. Click the **toolbar button** (graph icon)
4. Change values
5. Click **Apply** to test
6. Click **Save** to keep them permanently

## 🆘 Troubleshooting

If the icon is missing:
- confirm you are in Space Center
- confirm you are in Career Mode
- confirm the DLL is in `GameData\DynamicResourceGains\Plugins\`
